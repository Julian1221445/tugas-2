# tugas-2
README – Panduan Pengisian Formulir Pendaftaran Komunitas Teknologi

Formulir ini digunakan untuk mendaftar sebagai anggota Komunitas Teknologi. Silakan isi seluruh kolom dengan benar dan lengkap sesuai petunjuk berikut:

Bagian Nama

Nama Depan: Isi dengan nama depan Anda (maksimal 15 karakter). Contoh: Julian

Nama Belakang: Isi dengan nama belakang Anda (maksimal 15 karakter). Contoh: Syah

Bagian Kontak

Email: Masukkan alamat email aktif yang valid. Contoh: julian@gmail.com

Telepon: Masukkan nomor telepon tanpa spasi atau tanda baca, hanya angka. Contoh: 081234567890

Data Pribadi

Usia: Isi dengan angka antara 17 hingga 45 tahun.

Tanggal Lahir: Pilih tanggal lahir Anda sesuai kalender.

Jenis Kelamin: Pilih salah satu, Laki-laki atau Perempuan.

Minat Komunitas
Pilih satu atau lebih bidang teknologi yang Anda minati:

Web Development

Data Science

Mobile Development

Cyber Security

Portofolio dan Dokumen Pendukung

Link Portofolio: Masukkan alamat web portofolio Anda yang diawali dengan https://. Contoh: https://github.com/julian

Foto Normal: Unggah file foto diri yang jelas dan formal.

Pengiriman Formulir

Klik tombol “Kirim” untuk mengirim data.

Klik tombol “Ulangi” untuk menghapus semua isian dan memulai kembali.

Catatan:
Pastikan semua kolom yang wajib diisi (required) telah terisi dengan benar. Data Anda akan digunakan hanya untuk keperluan internal komunitas.
